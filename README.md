# Projeto DevOps 🚀

Aplicação desenvolvida com foco em práticas modernas de DevOps utilizando Docker, AWS e CI/CD.

---

# 📌 Sobre o projeto

Este projeto tem como objetivo demonstrar um fluxo completo de deploy de uma aplicação containerizada utilizando serviços da AWS e automações de infraestrutura.

O ambiente foi estruturado para simular um cenário real de produção, incluindo:

- Containerização com Docker
- Deploy automatizado
- Pipeline CI/CD
- Versionamento com GitHub
- Infraestrutura na AWS

# 🛠️ Tecnologias utilizadas

## Aplicação

- Next.js

## DevOps

- Docker

---

# 🐳 Docker

A aplicação foi containerizada utilizando Docker, permitindo:

- Padronização do ambiente
- Facilidade no deploy
- Melhor gerenciamento da aplicação
- Execução isolada em containers

---

# ☁️ AWS

Infraestrutura hospedada na AWS utilizando:

- EC2 para execução da aplicação
- ECR para armazenamento das imagens Docker
- IAM para permissões
- Security Groups para controle de acesso

---

# 🔄 CI/CD

Pipeline automatizada utilizando GitHub Actions para:

- Build da aplicação
- Criação da imagem Docker
- Push para o Amazon ECR
- Deploy automatizado na EC2

---
